# Task Quality Demo

Java-проект для практической работы по непрерывной интеграции и анализу качества кода.

## Что делает проект

Проект реализует небольшой сервис управления задачами:

- создание задачи;
- изменение статуса задачи;
- поиск задачи по идентификатору;
- фильтрация задач по статусу;
- получение статистики по задачам.

## Запуск локально

```bash
mvn clean test
mvn clean verify
```

## Интеграция с Travis CI и SonarCloud

1. Создать репозиторий на GitHub.
2. Загрузить проект в репозиторий.
3. В SonarCloud импортировать репозиторий GitHub.
4. В SonarCloud создать токен.
5. В Travis CI добавить переменную окружения `SONAR_TOKEN`.
6. Выполнить коммит в репозиторий и дождаться сборки.
